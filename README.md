# CodingGraceGame
A Python text-based adventure game for introductory programming courses. Three rooms, a Rock-Paper-Scissors mini-game, ASCII art, and a dictionary-driven player state illustrate core concepts including functions, loops, exception handling, input validation, and f-strings.

Based on the original text-based adventure game by [Coding Grace](https://www.codinggrace.com/), an Irish not-for-profit organization supporting diversity in tech. Source repository: [codinggrace/text_based_adventure_game](https://github.com/codinggrace/text_based_adventure_game).
